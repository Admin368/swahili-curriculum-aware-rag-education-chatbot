# Curriculum Aware Rag Education Chatbot

## Stack

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Drizzle](https://orm.drizzle.team)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)

- NodeJS

## Commands
### installing packages:
`bash
    pnpm install
`

### running development:
`
    pnpm dev
`

### running building:
`
    pnpm build
`

### running built version:
`
    pnpm start
`
